Detail van de methoden

addCourse

Met deze methode is het mogelijk om een vakken aan te maken in Smartschool. Er zijn 3 mogelijke parameters.

  • string $strCourseName: Vaknaam
  • string $strCourseDesc: Vakomschrijving
  • int $intVisibility: Zichtbaarheid (optioneel: '1' = zichtbaar (default) of '0' = niet zichtbaar)

Retourneert een boolean 'TRUE' als het vak succesvol is toegevoegd. Als de bewerking mislukt krijg je een string met een foutboodschap.

addCourseStudents

Met deze methode is het mogelijk om bepaalde klassen toe te voegen aan een bepaald vak. Er zijn 3 parameters.

  • string $strCourseName: Vaknaam
  • string $strCourseDesc: Vakomschrijving
  • string $strGroupIds: door komma gescheiden CSV-lijst met unieke klas- of groepscode(s)

Retourneert een boolean 'TRUE' als de leerlingen succesvol gekoppeld werden. Als de bewerking mislukt krijg je een string met een foutboodschap.

addCourseTeacher

Met deze methode is het mogelijk om bepaalde klassen toe te voegen aan een bepaald vak. Er zijn 3 parameters.

  • string $strCourseName: Vaknaam
  • string $strCourseDesc: Vakomschrijving
  • string $strLeerkrachtId: Uniek veld gebruiker die als hoofdleerkracht wordt toegevoegd
  • string $strCoLeerkrachtIds: door komma gescheiden CSV-lijst met uniek veld gebruiker die als co-leerkracht wordt toegevoegd

Retourneert een boolean 'TRUE' als de leerkracht(en) succesvol gekoppeld werd(en). Als de bewerking mislukt krijg je een string met een foutboodschap.
